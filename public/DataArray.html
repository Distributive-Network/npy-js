Constructor

# new DataArray(arrayBuffer, dtype, shape, header)

The data array class containes the typed array of the numpy file and can be converted to a shaped array by using the nj.array property. It also contains information about the dtype and the shape.

It is important that the shape and the datatypes are not changed. If they are, the header is incorrect and must be fixed.

The arraybuffer containing the data of the numpy array
